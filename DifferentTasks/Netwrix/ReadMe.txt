WinLogViewer 1.0.6.8


Данная программа предназначена для быстрой загрузки, фильтрации и поиска событий системных журналов. Обладает удобным и простым интерфейсом для просмотра, импорта и экспорта событий.

Системные требования

Windows 2000 SP2 or later
MS .Net Framework 3.5 SP4 or later
